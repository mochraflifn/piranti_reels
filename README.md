## Tugas Praktek

Flutter Instagram-like App using Unsplash API
Aplikasi Flutter sederhana yang menampilkan foto-foto dari Unsplash API dengan fitur scroll feed, like, comment, dan bookmark. Aplikasi ini terinspirasi dari tampilan dan interaksi aplikasi Instagram.

Fitur Utama
Scroll Feed: Menampilkan daftar foto dari Unsplash secara vertikal.
Like: Pengguna dapat menandai foto favorit.
Comment: Pengguna dapat menambahkan komentar pada setiap foto (disimpan secara lokal).
Bookmark: Pengguna dapat menyimpan foto untuk dilihat kembali.
API Integration: Menggunakan Unsplash API untuk mengambil data gambar secara real-time.
Responsive UI: Tampilan menyesuaikan ukuran layar perangkat.

Teknologi yang Digunakan
Flutter SDK
Dart
Unsplash API
HTTP package untuk komunikasi dengan REST API
Provider / Riverpod / GetX (sesuaikan dengan state management yang digunakan)
SharedPreferences / Hive untuk penyimpanan data lokal
